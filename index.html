ninedawg industries presents: bad paint

Instructions

This is a really bad version of paint.
Pressing 'Generate Grid' generates a square grid of boxes (you can specify the size!)

From there, you can click on each box to color it in. If you feel like it's not good, you can click the erase button to start over, or you can generate a new grid (possibly of a new size!)

Oh, also, if you're still bored, you can click here to play with more apps.
